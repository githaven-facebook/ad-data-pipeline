# Ad Data Pipeline

Airflow-based data pipeline for processing Facebook ad event data stored in S3.

## Overview

This service processes ad event data generated by the ad-event-stream service. It runs scheduled
Airflow DAGs that handle S3 data management, user segmentation, ranking features, and advertiser
reporting.

## Quick Start

```bash
make install
make docker-build
docker-compose up -d
```

See full documentation below for architecture details and deployment guide.
